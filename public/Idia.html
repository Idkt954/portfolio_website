Inexperienced drivers aiding system

I-dia camp

about

     This project was a group work I created with my friends during I-dia Camp. At first, we wanted to simulate a self-driving ambulance but we did not make it on time, so we decided to change it to this project instead. This project involves 2 functions which are 1. turn signal 2. object detection function.
     1. Turn Signal: There are 2 buttons that we put on the board. When the left one is pushed then the left light blinks. Likewise, when the right button is pressed the right light will blink.
     2. Object Detection: We put 2 HC-SR04 ultrasonic sensors on left and right side of the board (I think we should have put them at the front and at the back instead). When any objects get closer than 20 centimeters then the buzzer will make a loud noise.
